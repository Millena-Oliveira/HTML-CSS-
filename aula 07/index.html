Exemplo de Float com Imagem

Este é um texto de exemplo que envolve a imagem. Quando usamos float: left;, a imagem fica alinhada à esquerda, e o texto flui ao redor dela. Isso era muito usado em sites e blogs para deixar a leitura mais agradável.

Também é possível usar float: right; para colocar a imagem do outro lado. Embora hoje em dia se use muito mais flexbox e grid, ainda é útil saber usar o float para algumas situações.
